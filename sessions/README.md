# Sessions

Full prompt-and-action transcripts from each work session on like67.com.

## Format

Each session captures:

- **User prompts** — verbatim, in blockquotes.
- **Actions** — a short summary of what happened: files created or edited, commands run, decisions made. Reasoning is trimmed; the goal is to show the *what*, not relive the *why*.
- **Files touched** — at the bottom of each session, the list of files that changed.
- **Commands run** — terminal commands the user executed during the session, with brief explanations.
